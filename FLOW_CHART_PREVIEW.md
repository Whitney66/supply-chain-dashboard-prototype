## 📐 技术细节

### SVG 结构

```xml
<svg width="335" height="50" viewBox="0 0 335 50">
  <!-- 节点1: 起点 -->
  <rect x="0" y="10" width="80" height="30" rx="4" 
        fill="#dbeafe" stroke="#3b82f6" stroke-width="1.5"/>
  <text x="40" y="25" text-anchor="middle" fill="#1e40af">
    供应商
  </text>
  
  <!-- 连接线1 -->
  <line x1="80" y1="25" x2="95" y2="25" 
        stroke="#94a3b8" stroke-width="2"/>
  <polygon points="95,25 90,22 90,28" fill="#94a3b8"/>
  
  <!-- 节点2: 中间 -->
  <rect x="95" y="10" width="80" height="30" rx="4" 
        fill="#f1f5f9" stroke="#94a3b8" stroke-width="1.5"/>
  <text x="135" y="25" text-anchor="middle" fill="#475569">
    国际物流
  </text>
  
  <!-- ... 更多节点 ... -->
</svg>
```

### 响应式计算

```typescript
const nodeWidth = 80;       // 节点宽度
const nodeHeight = 30;      // 节点高度
const gap = 15;             // 节点间隙
const totalWidth = stepCount * nodeWidth + (stepCount - 1) * gap;
```

**4个节点的计算**:
- 总宽度 = 4 × 80 + 3 × 15 = 320 + 45 = **365px**
