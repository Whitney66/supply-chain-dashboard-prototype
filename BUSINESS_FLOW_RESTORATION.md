## 🎨 SupplyChainFlowChart 组件功能

### 组件文件
**路径**: `/src/app/components/SupplyChainFlowChart.tsx`

### 主要功能
展示完整的供应链物流业务流程，分为三个主要段落：

#### 1. 📦 订货段 (蓝色)
**流程**: 货品从下单到入一盘货仓库

```
[供应商] → [一线通关] → [提货运输] → [一盘货仓库]
  下单      报关放行      至海综保        入库
```

**包含指标**: 4个
- 供应商下单时效
- 一线通关时效  
- 提货运输时效
- 一盘货入库时效

**特殊链路**: 直发门店链路
```
[供应商] → [门店监管仓]
```

---

#### 2. 🚚 分货段 (绿色)
**流程**: 货品从一盘货仓库分配到门店监管仓

```
[仓库出库] → [二线通关] → [门店提货] → [门店上架]
  备货发运     清关放行      至监管仓      准备销售
```

**包含指标**: 4个
- 仓库出库时效
- 二线通关时效
- 门店提货时效
- 门店上架时效

---

#### 3. 🏪 门店段 (紫色)
**流程**: 货品从门店监管仓到达顾客手中

**主流程**:
```
[监管仓] → [周转仓] → [卖场] → [分拣仓] → [顾客收货]
  存储       调拨      陈列      分拣        完成
```

**包含指标**: 11个

**子链路详情**:

##### ① 常规销售链路 (4个指标)
```
监管仓 → 周转仓 → 卖场 → 分拣仓
```

##### ② 邮寄配送 (1个指标)
```
分拣仓 → 邮寄 → 顾客
```

##### ③ 提货点配送 (2个指标)
```
分拣仓 → 提货点 → 顾客
```

##### ④ 预定商品邮寄 (1个指标)
```
预定仓 → 邮寄 → 顾客
```

##### ⑤ 预定商品提货 (2个指标)
```
预定仓 → 提货点 → 顾客
```

##### ⑥ 预定仓调拨 (1个指标)
```
监管/周转仓 → 预定仓
```
